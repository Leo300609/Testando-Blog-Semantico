# 📖 Blog de Aventuras - HTML Semântico e Layout Flexbox

Este projeto é um blog estruturado com foco total em **Semântica HTML5** e layout moderno utilizando **Flexbox**. O objetivo foi criar uma página de notícias/artigos que seja acessível, organizada e fácil de manter.

## 🚀 Tecnologias Utilizadas
* **HTML5 Semântico:** Uso estratégico de tags como `<main>`, `<article>`, `<aside>`, `<nav>` e `<footer>` para melhor indexação e acessibilidade.
* **CSS3 Avançado:** * **Flexbox:** Sistema de colunas dinâmico para separar o conteúdo principal (`main`) da barra lateral (`sidebar`).
    * **Google Fonts & Tipografia:** Uso da fonte Georgia para um aspecto de "leitura de blog" clássica e elegante.
    * **Design Responsivo:** Implementação de Media Queries para garantir que o layout se adapte a telas menores.

## 🧠 Conceitos Praticados:
* **Estrutura Main-Aside:** Separação clara entre o conteúdo de maior relevância e as ferramentas auxiliares (busca, categorias e tags).
* **Componentização de UI:** Criação de "cards" de posts com imagens, títulos, metadados (autor) e descrições.
* **Estilização de Elementos de Navegação:** Tags em formato de "pills" e menus com efeitos de transição no hover.

## ⚙️ Como visualizar
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
